hexo+GitHub Pages设置

第一步：准备工作（环境安装）

在开始之前，确保你的电脑上已经安装了以下两个工具：

  1. Node.js（提供 Hexo 运行环境）
  2. Git（用于将代码推送到 GitHub）

安装完成后，打开终端（Windows 用户使用 Git Bash 或 CMD，Mac 用户使用 Terminal），运行以下命令检查是否安装成功：

第二步：在 GitHub 上创建仓库

  1. 登录你的 GitHub 账号。
  2. 点击右上角的 +，选择 New repository。
  3. 关键步骤：仓库名称（Repository Name）必须填写为： 你的GitHub用户名.github.io
  4. 设为 Public（公开）。
  5. 不要勾选 “Add a README file”，直接点击 Create repository。

第三步：配置 Git 与 GitHub 关联（免密上传）

打开 CMD / 终端，逐行执行命令：

  1. 设置 Git 用户名 / 邮箱（填你 GitHub 的信息）
  1. 生成 SSH 密钥（免密推送，必做）

连续按 3 次回车（不要设置密码）

  1. 复制公钥
    Windows：
    打开 C:\Users\你的用户名.ssh\id_ed25519.pub，复制全部内容

Mac/Linux：

复制输出的全部内容

  1. 把公钥粘贴到 GitHub
    1. GitHub → 右上角头像 → Settings
    2. 左侧 → SSH and GPG keys → New SSH key
    3. Title 随便填（如 Hexo），Key 粘贴刚才复制的内容
    4. 点击 Add SSH key

打开浏览器访问 http://localhost:4000 看到 Hexo 默认的欢迎页面就说明本地搭建成功了。按 Ctrl + C 可以关闭本地服务。

🚀 日常写博客的常用命令：
hexo new “文章标题” ：创建一篇新文章（在 source/_posts 目录下生成 .md 文件）。
hexo g ：写完后生成静态页。
hexo s ：在本地预览效果。
hexo d ：一键推送到线上。
